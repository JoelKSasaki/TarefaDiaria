# TarefaDiaria

1) Armazenei as tarefas e subtarefas como objetos, que serão automaticamente criados com o status FALSE, pois uma tarefa que acabou de ser criada não tem como ser dada como concluída instantaneamente. As subtarefas são opcionais. Se existirem, todas terão que ser cumpridas para concluir a tarefa principal.

2) Fiquei travado em relação a mostrar os elementos do(s) objeto(s) no HTML. Quando eu apertava o botão "Adicionar", os dados ficavam à mostra enquanto a página carregava e sumiam logo depois.

3) Eu implementaria os filtros e eu faria os botões de apagar a tarefa e marcá-la como concluída (eu implementaria também marcar as subtarefas como concluída).

4) Eu implementaria as funcionalidades que melhorariam sua experiência com a aplicação web.

5) Usei a IA para sanar algumas dúvidas sobre a linguagem JavaScript em si, pois este é meu primeiro projeto que uso JavaScript de verdade.
